# nil# Задание 1. Описание тематики дипломного проекта

## Тема: Разработка персонального веб-сайта музыкального артиста MZLFF с системой комментариев и оценок от аудитории


## 1. Актуальность темы

Музыкальная индустрия стремительно переходит в цифровое пространство. Сегодня наличие только страниц в социальных сетях уже недостаточно для артиста, который хочет иметь прямой контакт со своей аудиторией без посредников в лице алгоритмов и платформ.

Илья Мазеллов, известный как MZLFF, — один из самых ярких представителей новой волны российских артистов, сочетающий карьеру стримера (более 2 миллионов подписчиков на Twitch) и музыканта (миллионы прослушиваний, масштабные туры по России). У него есть лояльная аудитория, которая хочет взаимодействовать с ним напрямую: оставлять комментарии, ставить оценки, делиться впечатлениями о творчестве.

Однако вся обратная связь сейчас разрознена по чатам, комментариям в соцсетях и теряется в потоке информации. Возникает потребность в едином веб-ресурсе, который станет официальной визиткой артиста и одновременно платформой для сбора структурированной обратной связи от поклонников. Такой проект решает реальную проблему артиста и является востребованным в современной диджитал-среде.

## 2. Целевая аудитория

Платформа ориентирована на две группы пользователей:

**Артист (MZLFF / администратор сайта):**
- Получает готовый персональный сайт с биографией, дискографией и всеми необходимыми ссылками.
- Может просматривать комментарии от поклонников, отслеживать средний рейтинг.
- Может удалять нежелательные комментарии и управлять наполнением сайта (добавлять альбомы, редактировать биографию и ссылки).

**Поклонники (зрители и слушатели):**
- Получают единый источник достоверной информации об артисте.
- Могут выразить своё мнение через систему оценок (звёздочки) и оставить комментарий.
- Имеют быстрый доступ ко всем официальным ресурсам артиста через раздел полезных ссылок.

## 3. Предполагаемые функциональные возможности

Веб-приложение будет представлять собой сайт из четырёх основных страниц:

**Главная страница (Биография):**
- Фотография артиста, творческий псевдоним.
- Развёрнутая биография: от первых шагов в Minecraft-блогинге до участия в баттлах (SLOVO, RBL), выхода концептуальных альбомов «К маяку вселенной», «Phantasmagoria» и масштабных туров.
- Жанры: лирический рэп, альтернативный хип-хоп, поп-рэп.
- Средний рейтинг артиста (звёздочки), рассчитанный на основе оценок пользователей.

**Дискография:**
- Список основных альбомов и EP с обложками, названиями и краткими описаниями:
    - «2018 Year» (2018)
    - «Ничего плохого» (2019)
    - «К маяку вселенной» (2022)
    - «ADHD» (2025)
    - «Phantasmagoria» (2025)
- Ссылки на популярные треки.

**Полезные ссылки:**
- Официальные ресурсы артиста:
    - Twitch-канал
    - YouTube-канал
    - Telegram-канал
    - Страница ВКонтакте
    - Музыкальные платформы (Spotify, Яндекс Музыка, Apple Music)
- Ссылки отображаются в виде кликабельных кнопок или карточек с иконками.

**Комментарии:**
- Список комментариев от зарегистрированных пользователей.
- Каждый комментарий содержит имя пользователя, текст и оценку (звёздочки).
- Форма добавления комментария (доступна после регистрации).
- Комментарий появляется на странице сразу после отправки (без перезагрузки).
- Администратор может удалять нежелательные комментарии.

**Общие элементы:**
- Регистрация и авторизация пользователей.
- Навигационное меню для перехода между страницами.

## 4. Используемые технологии (предварительно)

- **Backend:** Python (Flask / Django) или Node.js (Express) — для обработки запросов, работы с базой данных и авторизации пользователей.
- **Frontend:** HTML5, CSS3, JavaScript. Для адаптивной вёрстки и современного внешнего вида — CSS-фреймворк Bootstrap.
- **База данных:** MySQL или PostgreSQL — для хранения информации о пользователях, комментариях и оценках.
- **Инструменты:** Figma (прототипирование интерфейса), Visual Studio Code (среда разработки), Git (контроль версий).

## 5. Ожидаемый результат

Результатом дипломного проекта станет готовый к использованию персональный веб-сайт артиста MZLFF. Он будет включать:

- Полноценную биографическую страницу с актуальной информацией об исполнителе.
- Структурированную дискографию с альбомами и описаниями.
- Раздел полезных ссылок на все официальные ресурсы артиста.
- Систему комментариев с возможностью добавления отзыва и оценки от зарегистрированных пользователей.
- Административную возможность управления комментариями (просмотр, удаление).

Продукт позволит артисту получить удобный инструмент для сбора обратной связи и презентации своего творчества, а поклонникам — официальную платформу для взаимодействия с любимым исполнителем. Проект имеет практическую ценность и может быть доработан до полноценного продакшн-решения.
